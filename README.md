# User Management POC

Full Stack User Management Application using Angular, Spring Boot, and PostgreSQL.

## Tech Stack

Frontend: Angular  
Backend: Spring Boot (REST API)  
Database: PostgreSQL  
Version Control: Git & GitHub

## Features

- Add User (First Name, Last Name, Pincode)
- Save data in PostgreSQL
- REST API integration
- Angular form validation

## Run Backend

1. Navigate to project folder
2. Run: mvn spring-boot:run
3. Server runs on: http://localhost:8080

## Run Frontend

1. Navigate to poc-ui folder
2. Run: npm install
3. Run: ng serve
4. Open: http://localhost:4200
